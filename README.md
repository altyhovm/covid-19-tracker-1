# Worldwide COVID-19 statistics.

### Website

https://covid-19.tracker.rowles.ch/

### API

https://covid-19-api.tracker.rowles.ch/statistics?page=1&limit=10

#### Querying the API

By country, e.g.:

`?country=United`

By province, e.g.:

`?province=Washington`

Paginate, e.g.:

`?page=2`, `?limit=25`
